# Pixso Tokens Sync

Синхронизация дизайн-токенов из Pixso в Kotlin.

## Файлы

| Файл | Описание |
|------|----------|
| `pixso_tokens_map.json` | Маппинг Pixso ID → Kotlin name |

## Как обновить токены

В Cursor скажи: **"обнови токены из Pixso"**

Система автоматически:
1. Получит токены из Pixso через MCP
2. Сравнит с `pixso_tokens_map.json`
3. Покажет изменения (добавлено/удалено/переименовано)
4. Обновит Kotlin файлы в `ui/theme/`

## Структура маппинга

```json
{
  "2:14": {
    "pixsoName": "Color/Primary/Primary_40",
    "kotlinName": "Color_Primary_Primary_40", 
    "type": "color",
    "key": "a06c56632740cd76f685ff01d820d35e5186e068"
  }
}
```

- **ID** (`2:14`) — уникальный ID переменной в Pixso
- **pixsoName** — имя в Pixso
- **kotlinName** — имя в Kotlin (сгенерированное)
- **type** — тип: `color`, `number`, `string`
- **key** — глобальный ключ для отслеживания между файлами

## Отслеживание изменений

| Изменение в Pixso | Как обнаруживается |
|-------------------|-------------------|
| Значение изменено | По ID — значение обновится |
| Токен переименован | По ID — предупреждение + обновление |
| Токен удалён | ID пропал — предупреждение |
| Токен добавлен | Новый ID — добавится |
